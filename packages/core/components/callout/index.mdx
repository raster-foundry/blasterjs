# Callout

## Basic usage

<Playground>
    <Callout>
        This is a basic callout. No title, icon, or other props.
    </Callout>
    <br />
    <Callout title="With title">
        Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod
        tempor incididunt ut labore et dolore magna aliqua.
    </Callout>
    <br />
    <Callout icon="calendar">
        With icon!
    </Callout>
    <br />
    <Callout icon="calendar" title="With icon and title">
        Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod
        tempor incididunt ut labore et dolore magna aliqua.
    </Callout>
</Playground>

## Intent

<Playground>
    <Callout intent={Intent.SUCCESS}>
        With intent. Note default icon for this intent (SUCCESS).
    </Callout>
    <br />
    <Callout title="With intent and title" intent={Intent.WARNING}>
        Note default icon for this intent (WARNING).
    </Callout>
    <br />
    <Callout icon={false} intent={Intent.WARNING}>
        With intent but icon=false. Lorem ipsum dolor sit amet, consectetur adipisicing
        tempor incididunt ut labore et dolore magna aliqua.
    </Callout>
    <br />
    <Callout icon="code" intent={Intent.DANGER}>
        With intent (DANGER) and custom icon, overriding intent's default icon.
    </Callout>
</Playground>


## Icon Size

## PropTypes

| Prop | Type | Required | Default | Description |
|:-----|:-----|:---------|:--------|:------------|
| Box props | | | | see [Box](/components/box) |
| ${space} | number or string | no | | see [styled-system](https://jxnblk.com/styled-system/) |
| intent | enum | no | `NONE` | see `common/intent` |
| icon | string or bool | no | | name of icon. overrides intent's default icon. |
| iconSize | string | no | | |
| title | string | no | | |
