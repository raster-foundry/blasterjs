# TextInput

Use `TextInput` for text-like input fields (`text, password, search, email, number,
url, tel`).

## Basic usage

<Playground>
    <TextInput placeholder="Your favorite movie" mb={4} />
    <TextInput placeholder="Your favorite dinner" disabled width="45%" mr={8} />
    <TextInput placeholder="Your favorite beverage" required width="45%" />
</Playground>
