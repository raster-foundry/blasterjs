# Input

Use `Input` for text-like input fields (`text, password, search, email, number,
url, tel`).

## Basic usage

<Playground>
    <Input placeholder="Your favorite movie" mb={4} />
    <Input placeholder="Your favorite dinner" disabled width="45%" mr={8} />
    <Input placeholder="Your favorite beverage" required width="45%" />
</Playground>
