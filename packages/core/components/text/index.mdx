# Text

## Import
`import { Text } from '@blasterjs/core'`

## Basic Usage

Composes [`Box`](/primitives/) and exposes text based prop functionality: `color`, `fontFamily`, `fontSize`, `fontWeight`, `textAlign`, `lineHeight`, `letterSpacing`.

The `<Text>` component renders a `<span>` with basic styles by default.

<Playground>
  <Text>Hello world</Text>
  <br />
  <Text tag="p">This is a paragraph</Text>
</Playground>
