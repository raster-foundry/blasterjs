### Usage

```jsx
<div>
  <Heading children="Content defined with a prop" />
  <Heading.h1 blue>This is a Heading.h1</Heading.h1>
  <Heading.h2 green>This is a Heading.h2</Heading.h2>
  <Heading.h3>This is a Heading.h3</Heading.h3>
  <Heading.h4>This is a Heading.h4</Heading.h4>
  <Heading.h5>This is a Heading.h5</Heading.h5>
  <Heading.h6>This is a Heading.h6</Heading.h6>
  <Heading alignment="center">This is a centered heading</Heading>
  {/*
  Alternative method:
  <Heading alignment={Alignment.CENTER}>This is a centered heading</Heading>
  */}
</div>
```
