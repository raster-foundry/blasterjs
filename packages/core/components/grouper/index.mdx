# Grouper

## Horizontal (default)

`COMPACT` density (default) forces intermediate `borderRadius` to `0`,
while `COMFORTABLE` density does not.

<Playground>
  <Grouper mb={4}>
    <Button>Hello</Button>
    <Button>Crazy</Button>
    <Button>World</Button>
  </Grouper>

  <Grouper density={Density.COMFORTABLE}>
    <Button>Hello</Button>
    <Button>Crazy</Button>
    <Button>World</Button>
  </Grouper>
</Playground>

## Gutter

  <Grouper gutter={2} mb={8}>
    <Button>Hello</Button>

  </Grouper>

## Radius override

`borderRadius` prop affects outer corners of first & last children only.
Ignored if `COMFORTABLE`.

<Playground>
  <Grouper borderRadius="large">
    <Button>Hello</Button>
    <Button>Crazy</Button>
    <Button>World</Button>
  </Grouper>
</Playground>


## Form

### PropTypes
| Prop | Type | Required | Default | Description |
|:-----|:-----|:---------|:--------|:------------|
| Box props | | | | see [Box](/components/box) |
| flexDirection | string | no | `row` | |
| direction | enum | no | | overrides `flexDirection`. see [Direction](/common/direction) |
| density | enum | no | `Density.COMPACT` | if COMPACT, interior borderRadii forced to 0. see [Density](/common/density) |
| gutter | string or number | no | `2px` if `COMPACT`. `space.1` if `COMFORTABLE` | gap between children |
| borderRadius | string | no | `base` | outer corners of first & last children. ignored if `COMPACT`. |
