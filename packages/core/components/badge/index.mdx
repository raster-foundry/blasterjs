| ${color} | string | false | see [styled-system COLOR](https://github.com/jxnblk/styled-system/blob/master/docs/api.md#color) |
| fontSize | number or string | false | see [styled-system FONTSIZE](https://github.com/jxnblk/styled-system/blob/master/docs/api.md#fontsize) |
| borderRadius | number or string | false | see [styled-system BORDERRADIUS](https://github.com/jxnblk/styled-system/blob/master/docs/api.md#borders) |
