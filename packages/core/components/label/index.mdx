# Label

## Import
`import { Label } from '@blasterjs/core'`

## Basic usage

<Playground>
    <Label>
        What is the airspeed velocity of an unladen swallow?
        <input type="text" />
    </Label>
</Playground>

<Playground>
    <Label htmlFor="swallow">What do you mean?</Label>

        <option>European</option>
    </Select>
</Playground>
