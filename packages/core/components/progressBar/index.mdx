# ProgressBar

## Import
`import { ProgressBar } from '@blasterjs/core'`

## Basic Usage

Displays a fractional amount of progress towards some end.

<Playground>
  <ProgressBar value={25} max={100} />
</Playground>

## Striped and animated
Use the `striped` and `animated` prop to change the default appearance and show motion.

<Playground>
  <ProgressBar 
    value={25} 
    max={100} 
    striped 
    animated 
  />
</Playground>

## Styling

<Playground>
  <ProgressBar 
    value={25} 
    max={100} 
    striped 
    animated
    color="greenShade"
    bg="greenTint"
  />
</Playground>

### ProgressBar PropTypes

| Prop | Type | Required | Default | Description |
|:-----|:-----|:---------|:--------|:------------|
| Box props | | | | see [Box](/components/box) |
| value | number | no | `0` | current progress value |
| max | number | no | `1` | max progress value |
| color | string | no | `primary`  | fill color |
| bg | string | no | `grayLightest` | track color |
| borderRadius | string or number | no | `999` | |
| width | string | no | `100%` | |
| height | string | no | `0.8rem` | |
| striped | boolean | no | false | |
| animated | boolean | no | false | animates the striped variant |
