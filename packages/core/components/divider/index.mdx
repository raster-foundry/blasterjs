# Divider

## Import
`import { Divider } from '@blasterjs/core'`

## Basic Usage

By default, the `Divider` component Composes [`Box`](/primitives/) with a default prop `tag=hr` which will render a horionzal rule.

<Playground>
  <Divider />
</Playground>

## Vertical Divider

<Playground>

  />

<Playground>

## PropTypes
| Prop | Type | Required | Default | Description |
|:-----|:-----|:---------|:--------|:------------|
| Box props | | | | see [Box](/components/box) |
| direction | enum | yes | `horizontal` | see [Direction](/common/direction) |
| color | string | no | | Theme color or valid css color value  |
| width | string | no | | Theme width or valid css width value |
| height | string | no | | Theme height or valid css height value |
