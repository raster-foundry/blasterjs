### PropTypes
Accepts standard html `<img>` attributes

| Prop | Type | Required | Description |
|:-----|:-----|:---------|:------------|
| rounded | bool | false | Adds a border-radius |
| circle | bool | false | Adds 100% border-radius |
| ${space} | string | false | see [styled-system SPACE](https://jxnblk.com/styled-system/#margin--padding) |
