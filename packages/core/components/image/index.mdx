### PropTypes
Accepts standard html `<img>` attributes

| Prop | Type | Required | Default | Description |
|:-----|:-----|:---------|:--------|:------------|
| ${space} | number or string | no |  | see [styled-system](https://jxnblk.com/styled-system/) |
| src | url | yes | | |
| width | string | no | auto | |
| height | string | no | auto | |
| shape | enum | no | SQUARE | see common/shape |
| borderRadius | number or string | no | | overrides shape |
