# Card

## Import
`import { Card } from '@blasterjs/core'`

## Basic Usage

Composes a [`Box`](/typography/box) with preset styles.

<Playground>
  <Card>
    Hello World
  </Card>
</Playground>

## Card density

`Card` exposes the prop `density`, which applies a consistent padding.

<Playground>
  <Card density="comfortable">
    Feels pretty comfortable in here
  </Card>
  <br />
  <Card density="compact">
    Feels a little compact in here
  </Card>
</Playground>

## Card with content

<Playground>
  <Card density="comfortable">
    <Header tag="h4">Hello world</Header>

    <Button appearance="prominent">Continue</Button>
    <Button appearance="minimal">Cancel</Button>
  </Card>
</Playground>

### PropTypes
| Prop | Type | Required | Default | Description |
|:-----|:-----|:---------|:--------|:------------|
| Box props | | | | see [Box](/primitives/) |
| density | enum | no |  | see [Density](/common/density) |
| bg | string | no | `white` | |
| border | string | no | `2px solid` | |
| borderColor | string | no | `grayLight3` | |
| borderRadius | string | no | `large` | |
| boxShadow | number or string | no | `2` | |
