# Header

## Import
`import { Header } from '@blasterjs/core'`

## Basic Usage

Composes [`Text`](/typography/text) and renders an `<h1>` by default.

<Playground>
  <Header>Header 1</Header>
</Playground>

## Changing header tag

Use the `tag` prop to change from the default `h1` element to `h2`, `h3`, `h4`, `h5`, or `h6`.

<Playground>
  <Header tag="h1">Header 1</Header>
  <Header tag="h2">Header 2</Header>
  <Header tag="h3">Header 3</Header>
  <Header tag="h4">Header 4</Header>
  <Header tag="h5">Header 5</Header>
  <Header tag="h6">Header 6</Header>
</Playground>

## Styling headers

<Playground>
  <Header color="primary" lineHeight="1" fontWeight="100">Header 1</Header>
  <Header textAlign="center">Center aligned header</Header>
</Playground>

### PropTypes

| Prop | Type | Required | Default | Description |
|:-----|:-----|:---------|:--------|:------------|
| Text props | | | | see [Text](/typography/text) |
| tag | string | no | `h1` | `[h1, h2, h3, h4, h5, h6]` |
| color | string | no | `textBase` | |
| lineHeight | number or string | no | `2` | |
